# SCP Consumer App

Mobile application for Restaurants and Hotels (B2B Consumers) to discover suppliers, place orders, and manage business relationships.

## Quick Start

```bash
# Install dependencies
flutter pub get

# Generate localizations
flutter gen-l10n

# Run the app
flutter run

# Build for production
flutter build apk --release
```
